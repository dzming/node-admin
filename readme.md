# Nodejs 项目

## 后台管理系统

> 用 node 实现一个电商网的后台管理系统

### 项目情况-运用技术

- 后端语言：使用的是 Nodejs+Express
- 数据库：使用的是 MongoDB
- UI 框架：使用 bootstrap 框架
- 登陆使用 token 进行加密、解密、验证

### 完成情况-功能实现

- 数据查询
  - 数据渲染
- 商品管理
  - 列表展示
  - 添加
  - 编辑
  - 删除
- 用户管理（管理员）
  - 添加
  - 编辑
  - 删除
- 用户管理（普通会员）
  - 修改用户信息（增删查改）
- 订单管理
  - 订单展示
  - 订单状态修改
    - 未付款
    - 付款成功
    - 发货
    - 签收
